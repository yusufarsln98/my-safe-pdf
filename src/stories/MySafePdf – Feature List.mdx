# 📄 Local PDF Tools – Feature List

## 🎯 Purpose

To provide a privacy-friendly, fully offline browser-based tool that allows users to work with **PDFs and images** (JPG/PNG) directly in the browser—no uploads, no servers.

## 🔁 User Flow

1. **Choose a module**: (Merge, Split, Edit, Images to PDF, etc.)
2. **Upload files**: PDF or JPG/PNG images
3. **Perform actions**: reorder, delete, rotate, split, etc.
4. _(Optional)_ Preview pages or images
5. **Download final PDF**: All done locally, no internet needed
